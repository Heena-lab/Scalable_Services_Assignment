# Student Service

Handles student CRUD, vaccination marking, and CSV import for the School Vaccination Portal.

## Endpoints
- `GET /students`
- `GET /students/:id`
- `POST /students`
- `PUT /students/:id`
- `PATCH /students/:id/vaccinate`

## Setup to run APIs with Postman
```bash
npm install
npm start
```
